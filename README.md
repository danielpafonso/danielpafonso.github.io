## Quote formation

| Field      | Format     | Required |
| ---------- | ---------- | -------- |
| date       | YYYY-MM-DD | yes      |
| quote      | string     | yes      |
| author     | string     | yes      |
| birth year | integer    | yes      |
| death year | integer    | no       |
| profession | string     | no       |

```
date|quote|author - birth_year-death_year, profession
```
